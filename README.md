올리브영 상품 크롤러
올리브영 웹사이트에서 상품 이미지와 정보를 자동으로 수집하는 파이썬 기반 크롤링 프로그램입니다.

🔎기능

- 올리브영 웹사이트에서 상품 이미지(url) 수집
- 상품명, 가격, 상세 정보 추출
- 수집된 데이터를 JSON 아래의 형식으로 저장
- "productName": 제품명 
  "price": 가격
  "thumbnailUrls": 썸네일 이미지
  "detailImageUrls": 상품 상제 이미지 url
  "manufacturer": 제조사
  "country0f0rigin": 제조국
  "productUrl": 상세페이지 url
